# Инструкция по запуску n8n локально

## Вариант 1: Через Docker (рекомендуется)

### Шаг 1: Установите Docker Desktop

1. Скачайте Docker Desktop для macOS: https://www.docker.com/products/docker-desktop/
2. Установите приложение
3. Запустите Docker Desktop (должна появиться иконка в строке меню)

### Шаг 2: Запустите n8n

После установки Docker выполните в терминале:

```bash
docker-compose up -d
```

### Шаг 3: Откройте n8n в браузере

n8n будет доступен по адресу: **http://localhost:5678**

**Учетные данные для входа:**
- Логин: `admin`
- Пароль: `admin`

### Остановка n8n

```bash
docker-compose down
```

### Просмотр логов

```bash
docker-compose logs -f n8n
```

---

## Вариант 2: Через npm (без Docker)

### Шаг 1: Установите Node.js

Установите Node.js через Homebrew:

```bash
brew install node
```

Или скачайте с официального сайта: https://nodejs.org/

### Шаг 2: Установите n8n глобально

```bash
npm install n8n -g
```

### Шаг 3: Запустите n8n

```bash
n8n start
```

n8n будет доступен по адресу: **http://localhost:5678**

**Примечание:** При использовании npm версии вам нужно будет вручную настроить переменные окружения и пути к данным, которые в Docker версии настроены автоматически.

---

## Рекомендация

Используйте **Вариант 1 (Docker)**, так как:
- Все настройки уже сконфигурированы в `docker-compose.yml`
- Данные сохраняются в volumes
- Легко обновлять версию n8n
- Изолированная среда

## Проверка установки Docker

После установки Docker Desktop проверьте:

```bash
docker --version
docker-compose --version
```

Оба команды должны вернуть версии установленных программ.
